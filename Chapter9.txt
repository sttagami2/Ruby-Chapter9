FizzBuzzプログラムとは
・元々はシンプルなパーティゲーム
・FizzBuzzができるかできないかでプログラマーの技術力が分かる
・コーディングの面接の題材としてよく使われる

FizzBuzzの内容
・「3」で割り切れる数値を引数に渡すと、「Fizz」を返す
・「5」で割り切れる数値を引数に渡すと、「Buzz」を返す
・「3」と「5」の両方で割り切れる数値を引数に渡すと、「FizzBuzz」を返す
・それ以外の数値は、その数値を文字列に変えて返す

メソッドを定義する　← 関数？
・メソッドを定義することで自分で作成したプログラムを簡易に実行できるようになる

	def メソッド名(引数)	← 引数は任意
		実行する処理内容
	end
例
	def hello_world
		"Hello, World!"
	end
	puts hello_world		→	Hello, World!


メソッドを定義するときは、「引数」と「戻り値」を理解しておく必要がある

ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー
ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー
引数とは
・メソッドを呼び出す際に、メソッドに渡す値のこと
・引数の値は好きな変数名をつけられる

	def greeting(name)
		"Hello, #{name}!"
	end
	puts greeting("John")	← Johnが引数でメソッドのnameに渡す値
ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー
ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー
戻り値とは
・メソッドが呼ばれた時に返ってくる値のこと
・def ~ end処理内の最終結果が戻り値になる
	def greeting
		"Hello, John!"
		"Hi, John!"
	end
	puts greeting		→	Hi, John!

returnによる戻り値
Returnを使用することで、戻り値を指定することもできる
